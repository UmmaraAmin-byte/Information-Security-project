SQL INJECTION MECHANICS

What is happening: The app concatenates user input directly into a SQL string. A single quote ' breaks out of the string context, letting the attacker write their own SQL.

Classic attacks: ' OR '1'='1 returns all rows. admin'-- bypasses password check. UNION-based attacks append extra SELECT to exfiltrate other tables.

Real impact: Heartland Payment Systems (2008) — 130 million credit cards stolen via SQLi. Yahoo (2012) — 450,000 credentials dumped. SQLi is OWASP Top 10 #3.

Why parameterized queries work: The database driver handles escaping internally. The ? placeholder is a bound parameter — the database never interprets it as SQL syntax.
